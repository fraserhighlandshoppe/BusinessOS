## Implementation Sequence:
1. **Initialize Pixel Code** – Insert standard pixel script on website
2. **Configure Events** – Track page views, add-to-cart, purchases
3. **Create Custom Audiences** – Build audiences from specific events
4. **Implement Retargeting** – Run ads to pixel visitors
5. **Measure Conversions** – Track ROI using pixel data
6. **Optimize Performance** – Adjust campaigns based on pixel insights

## Key Principles:
- **Event Tracking** – Capture all relevant user interactions
- **Audience Segmentation** – Build custom audiences from pixel events
- **Retargeting Efficiency** – Use pixel data for precise ad targeting
- **Data-Driven Decisions** – Adjust bids and budgets based on performance
- **Compliance** – Ensure proper consent and privacy settings

## Checklist:
- [ ] Pixel code installed on all relevant site pages
- [ ] Events configured for key actions (view content, add to cart, purchase)
- [ ] Custom audiences created for each event
- [ ] Retargeting campaigns set up with appropriate bids
- [ ] Conversion metrics dashboard active
- [ ] Performance alerts configured (e.g., drop in conversion rate)

## Metrics & KPIS:
- **Purchase Conversion Rate**: Target 2-5% of retargeted visitors
- **Cost Per Acquisition (CPA)**: Optimize to stay within target CPA
- **ROAS (Return on Ad Spend)**: Maintain >3x for pixel-driven campaigns
- **Pixel Firing Rate**: Ensure >95% tracking accuracy
- **Audience Quality**: Monitor engagement metrics of retargeted segments
- **Revenue Attribution**: Track revenue generated from retargeting ads
