## Implementation Sequence:
1. **Keyword Research** – Build core keyword list using competitor analysis
2. **Ad Group Structuring** – Organize by theme and intent level
3. **Ad Copy Creation** – Write compelling ads using proven formulas
4. **Landing Page Alignment** – Ensure message match from ad to page
5. **Bid Management** – Set initial bids, adjust based on performance
6. **Quality Score Optimization** – Improve relevance and expected CTR
6. **Split Testing** – Apply [[Split_Testing_Framework]] for continuous improvement

## Key Principles:
- **Relevance First** – Keywords, ads, and landing pages must align
- **Negative Keywords** – Aggressively filter irrelevant traffic
- **Ad Extensions** – Maximize SERP real estate with sitelinks, callouts
- **Device Targeting** – Optimize separately for mobile vs desktop
- **Dayparting** – Schedule ads for peak performance hours
- **Geo-Targeting** – Focus budget on high-converting locations

## Checklist:
- [ ] Core keyword list >50 terms researched
- [ ] Ad groups structured by match type and intent
- [ ] Minimum 3 ad variations per ad group
- [ ] Negative keyword list >100 terms
- [ ] Ad extensions configured (sitelinks, callouts, structured snippets)
- [ ] Conversion tracking validated in Google Ads
- [ ] Quality Score monitoring established
- [ ] Bid adjustment rules documented

## Metrics & KPIs:
- **Click-Through Rate (CTR)**: Target >2% for search, >0.5% for display
- **Quality Score**: Maintain >7/10 on core keywords
- **Cost Per Click (CPC)**: Decreasing trend month-over-month
- **Conversion Rate**: Target >3% for lead gen, >1% for e-commerce
- **Cost Per Acquisition (CPA)**: Below target threshold
- **Return on Ad Spend (ROAS)**: >400% for profitable campaigns
- **Search Impression Share**: >50% on brand terms
