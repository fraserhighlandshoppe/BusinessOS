# Traffic Generation System - Digital Marketer

## Overview
Build or grow any business with just $10/day in paid traffic. The system divides budget across three traffic temperatures:

- **$6/day (60%)**: Cold traffic (new audience introduction)
- **$3/day (30%)**: Warm traffic (engaged prospects)
- **$1/day (10%)**: Hot traffic (existing customers)

## Section 4: 6 Step Process for Creating Content to Cold Traffic

### Step 1: Write Backwards
- Start with the call-to-action first
- Work backwards to create supporting content
- Purpose: Get them to take a specific action

### Step 2: Choose the Topic
Answer these questions:
- What does your target need to know before converting?
- What barriers prevent conversion?
- What objections do they have?
- What's their primary pain point?

### Step 3: Pick a Post Type
Common types:
- List of things/points
- How-to post
- Content aggregator (summarizing others' content)
- Crowdsource (expert opinions on same question)
- Embed reactor (reaction to viral video)

Resources:
- The Ultimate List of Blog Post Ideas
- The Content Engine Execution Plan

### Step 4: Edit for Consumption
- Headline: Clear, not cute (10-12 words max)
- First sentence: Short, punchy, curiosity-based
- Transitions: Between paragraphs/subheaders
- Formatting: Bullets, numbered lists, block quotes
- Paragraph length: ~3 sentences each
- Peaks and valleys: Alternate wide/narrow text sections

### Step 5: Use the "Deep Tease"
- Optional curiosity-based tease deep in content
- Place near top with reference to later section
- Example: "Note method 5 requires..."

### Step 6: Insert 2 Types of CTA
1. **Text-based CTA** - Text link (people expect to click text)
2. **Graphic CTA** - Image with button (TV commercial style)
   - Include visible button on graphic
   - Alternative: Johnson Box (HTML code box with links)
