# 5-Step Sales Funnel Framework

## Core Philosophy
> "A conversion funnel is a multi-step, multi-modality campaign that seamlessly and subtly leads a prospect toward a desired action. The 5-step process provides the systematic structure."

This framework provides the detailed 5-step process that underlies all funnel types (Acquisition, Activation, Monetization).

## The 5 Steps

### Step 1: Lead Magnet (Entry Point)
**Purpose**: Exchange value for contact information

**Characteristics**:
- Free, high-value resource
- Solves specific, immediate problem
- Consumable in 5-10 minutes
- Demonstrates expertise

**Examples**:
- Blog post template
- Checklist/cheatsheet
- Short video training
- PDF guide/whitepaper
- Tool/calculator

**Metrics**:
- Opt-in rate: Target 20-40%
- Cost per lead (CPL)
- Lead quality score

### Step 2: Tripwire Offer (Low-Risk Purchase)
**Purpose**: Convert leads into buyers with minimal friction

**Characteristics**:
- Low price point ($7-$47)
- High perceived value (85%+ discount)
- Directly related to lead magnet
- Immediate delivery
- No-brainer purchase decision

**Examples**:
- $7 execution plan (regular $47)
- $27 mini-course (regular $197)
- $37 template pack (regular $197)
- 14-day trial for $1

**Metrics**:
- Conversion rate: Target 3-10%
- Average order value (AOV)
- Tripwire-to-core conversion rate

### Step 3: Core Offer (Main Product/Service)
**Purpose**: Deliver flagship product at full price

**Characteristics**:
- Primary revenue generator
- Solves complete problem
- Higher price point ($97-$2,000+)
- Includes ongoing support/access
- Clear transformation promise

**Examples**:
- Membership site ($38.60/month)
- Flagship course ($997)
- Done-for-you service ($2,500)
- Software subscription ($197/month)

**Metrics**:
- Conversion rate: Target 1-5%
- Customer lifetime value (CLV)
- Refund rate: < 5%

### Step 4: Profit Maximizer (Upsells/Cross-sells)
**Purpose**: Increase average customer value

**Characteristics**:
- Complementary to core offer
- Higher margin
- Presented immediately after purchase
- One-click purchase
- Logical next step

**Types**:
- **Order Bump**: Small add-on at checkout ($27-$97)
- **Upsell 1**: Enhanced version ($197-$997)
- **Upsell 2**: Done-for-you/implementation ($997-$5,000)
- **Cross-sell**: Related product ($47-$297)
- **Continuity**: Recurring revenue ($27-$197/month)

**Metrics**:
- Take rate per upsell: Target 15-30%
- Average customer value (ACV)
- Profit maximizer revenue %

### Step 5: Return Path (Retention & Ascension)
**Purpose**: Convert buyers into multi-buyers and advocates

**Characteristics**:
- Ongoing engagement
- Regular value delivery
- Ascension offers
- Referral/advocacy program
- Win-back campaigns

**Strategies**:
- Email nurture sequences
- Content marketing
- Community access
- Loyalty programs
- Advanced training/coaching
- Affiliate/partner programs

**Metrics**:
- Repeat purchase rate
- Referral rate
- Net promoter score (NPS)
- Churn rate (for continuity)

## Funnel Math - Key Equations

### Customer Acquisition Cost (CAC)
```
CAC = Total Ad Spend / Total New Customers
```

### Average Customer Value (ACV)
```
ACV = (Tripwire Revenue + Core Revenue + Upsell Revenue + Continuity Revenue) / Total Customers
```

### Return on Ad Spend (ROAS)
```
ROAS = ACV / CAC
Target: ROAS > 3:1 (break-even at 1:1)
```

### Funnel Conversion Rates
```
Lead Magnet Opt-in Rate = Leads / Visitors
Tripwire Conversion = Tripwire Buyers / Leads
Core Conversion = Core Buyers / Tripwire Buyers
Upsell Take Rate = Upsell Buyers / Core Buyers
```

## Funnel Architecture Principles

### 1. Sequential Trust Building
Each step reduces risk and builds trust for the next:
1. Free value (Lead Magnet) → Trust established
2. Low risk purchase (Tripwire) → Buyer identity
3. Full solution (Core) → Transformation delivered
4. Enhanced value (Profit Maximizers) → Maximum value
5. Ongoing relationship (Return Path) → Lifetime value

### 2. Congruency Rule
Every step must be logically connected:
- Lead Magnet topic → Tripwire topic → Core topic
- No jarring transitions
- Each step promises the next

### 3. Speed to Value
- Lead Magnet: Immediate
- Tripwire: < 5 minutes
- Core: Progressive delivery
- Profit Maximizers: Instant access

### 4. Automation First
- All steps automated where possible
- Email sequences for each transition
- Retargeting for drop-offs
- Behavioral triggers

## Traffic Temperature Alignment

| Funnel Step | Traffic Temperature | Message Match |
|-------------|---------------------|---------------|
| Lead Magnet | Cold | Problem-aware |
| Tripwire | Cold/Warm | Solution-aware |
| Core | Warm/Hot | Product-aware |
| Profit Maximizers | Hot | Most-aware |
| Return Path | Hot | Brand-loyal |

## Implementation Checklist

### Phase 1: Design (Week 1)
- [ ] Map customer journey
- [ ] Define Lead Magnet topic
- [ ] Design Tripwire offer
- [ ] Outline Core Offer
- [ ] Plan Profit Maximizers
- [ ] Design Return Path

### Phase 2: Build (Weeks 2-4)
- [ ] Create Lead Magnet
- [ ] Build Tripwire sales page
- [ ] Build Core Offer sales page
- [ ] Create Upsell pages
- [ ] Set up email sequences
- [ ] Configure payment processing
- [ ] Set up tracking/analytics

### Phase 3: Launch & Optimize (Weeks 5-8)
- [ ] Launch to warm traffic first
- [ ] Test cold traffic
- [ ] Optimize each step conversion
- [ ] Add retargeting
- [ ] Scale winning elements

## Common Mistakes

1. **Skipping Steps**: Going straight to Core Offer
2. **Incongruent Offers**: Unrelated Tripwire to Core
3. **No Profit Maximizers**: Leaving 40%+ revenue on table
4. **Weak Return Path**: No repeat purchase system
5. **Poor Tracking**: Can't optimize what you don't measure
6. **All Cold Traffic**: Not segmenting by temperature
