# Digital Marketer 10-Point Blog Post Audit

## Core Philosophy
Every blog post falls into one of three reactions: magnificent, mediocre, or dreadful. The 10-Point Blog Post Audit provides a systematic process to transform "meh" posts into magnificent content by evaluating 10 specific elements.

## The 10-Point Audit Framework

### The 10 Elements to Audit
1. **Headline** - Promise clarity and delivery
2. **Introduction** - Hook and context setting
3. **Consumption** - Readability and scannability
4. **Goal** - Clear purpose and objective
5. **Media** - Supporting visuals and multimedia
6. **Close** - Conclusion and call-to-action
7. **Search** - SEO optimization
8. **Categorization** - Proper taxonomy
9. **Completeness** - Full delivery on promise
10. **Clarity** - Clear, understandable communication

## Detailed Audit Elements

### 1. Headline
**Exceptional Criteria**:
- Clear, tightly-written promise using necessary words
- Promise delivered upon in post body
- Compelling without being misleading or "hypey"
- Formula: "How to Get [DESIRED END RESULT] in [DESIRABLE AMOUNT OF TIME]"

### 2. Introduction
**Exceptional Criteria**:
- Grabs attention in first sentence
- Sets context and explains relevance
- Creates curiosity about what's coming
- Connects emotionally or intellectually with reader

### 3. Consumption
**Exceptional Criteria**:
- Easy to read and scan
- Proper use of subheadings, bullet points, whitespace
- Logical flow and organization
- Mobile-friendly formatting

### 4. Goal
**Exceptional Criteria**:
- Clear, single purpose stated early
- Aligns with business objectives
- Reader knows what they'll learn/achieve
- Measurable outcome defined

### 5. Media
**Exceptional Criteria**:
- Relevant, high-quality images/video/audio
- Proper attribution and sizing
- Enhances understanding (not just decoration)
- Alt text for accessibility and SEO

### 6. Close
**Exceptional Criteria**:
- Clear conclusion summarizing key points
- Strong call-to-action (CTA)
- Next steps or resources provided
- Reinforces the post's value proposition

### 7. Search (SEO)
**Exceptional Criteria**:
- Target keyword in title tag
- Keyword naturally included in body
- Keyword in image alt attributes
- Keyword in URL and meta description
- Related: How to Do On-Page SEO

### 8. Categorization
**Exceptional Criteria**:
- Post in appropriate category
- Properly tagged with relevant keywords
- Intuitive category names for audience navigation
- Consistent taxonomy usage

### 9. Completeness
**Exceptional Criteria**:
- Completely delivers on headline/introduction promise
- Every idea appropriately "fleshed out"
- Includes supporting media, examples, data, links
- Goes the extra mile rather than leaving audience wanting

### 10. Clarity
**Exceptional Criteria**:
- Consistent with brand voice and tone
- Aligns with other organizational communications
- No contradictions with previously published content
- Clear, jargon-free language appropriate for audience

## Audit Process

### Step 1: Select Post for Audit
- Choose a recent or upcoming blog post
- Have full content ready for review

### Step 2: Grade Each Element
Use this scoring system for each of the 10 elements:
- **Exceptional** (9-10 points): Exceeds expectations
- **Proficient** (7-8 points): Meets expectations
- **Developing** (5-6 points): Approaching expectations
- **Beginning** (3-4 points): Below expectations
- **Unsatisfactory** (0-2 points): Far below expectations

### Step 3: Calculate Final Score
- Total possible: 100 points (10 elements × 10 points each)
- **90-100**: Magnificent (Minor tweaks only)
- **80-89**: Strong (Several improvements needed)
- **70-79**: Satisfactory (Moderate revisions required)
- **60-69**: Needs Work (Significant improvements)
- **Below 60**: Major Overhaul Required

### Step 4: Create Action Items
For each element scoring below 9:
- Identify specific improvement needed
- Create actionable task
- Set deadline for completion

### Quick Audit Checklist
Before publishing, ask:
1. [ ] Does headline make clear promise delivered in body?
2. [ ] Does introduction hook reader and set context?
3. [ ] Is content easy to scan with proper formatting?
4. [ ] Is goal/purpose clear from the beginning?
5. [ ] Is media relevant, high-quality, and attributed?
6. [ ] Does close summarize and provide clear CTA?
7. [ ] Is target keyword used for SEO?
8. [ ] Is post properly categorized and tagged?
9. [ ] Does content fully deliver on promised value?
10. [ ] Is tone consistent with brand and other content?

## Usage Guidelines

### When to Use
- **Before publishing**: Final quality check
- **After publishing**: Performance analysis and improvement
- **Content planning**: Template for new posts
- **Team training**: Editorial standard setting
- **Client work**: Agency deliverable quality assurance

### Frequency Recommendations
- **Individual posts**: Audit every piece before publishing
- **Batch content**: Audit 10% sample for quality control
- **Team content**: Weekly audit rotation for consistency
- **Client deliverables**: Audit before submission

### Time Investment
- **Quick audit**: 5-10 minutes per post
- **Deep audit**: 15-20 minutes per post
- **Team training**: 60-minute workshop
- **Monthly review**: 30-minute team session

## Improvement Workflow

### Low-Scoring Elements Action Plan
**If Headline scores <7**:
- Rewrite using proven formulas
- Test multiple variations
- Ensure promise matches content

**If Introduction scores <7**:
- Start with question, statistic, or story
- Clearly state what reader will learn
- Match tone to audience expectations

**If Consumption scores <7**:
- Add subheadings every 2-3 paragraphs
- Use bullet points for lists
- Increase whitespace and paragraph breaks
- Check mobile rendering

**If Goal scores <7**:
- Add clear objective statement in first paragraph
- Align with business/marketing goals
- Make outcome measurable and specific

**If Media scores <7**:
- Add at least one relevant image
- Create custom graphics when possible
- Optimize images for web (compression, dimensions)
- Add descriptive alt text

**If Close scores <7**:
- Summarize 3 key takeaways
- Add clear, specific CTA
- Provide next steps or resources
- Link to related content

**If Search scores <7**:
- Research and select target keyword
- Optimize title tag (under 60 characters)
- Include keyword in first 100 words
- Add keyword to image alt text
- Optimize meta description (under 160 characters)

**If Categorization scores <7**:
- Select most specific relevant category
- Use 3-5 relevant tags
- Follow established taxonomy
- Avoid overly broad or vague tags

**If Completeness scores <7**:
- Add supporting examples and case studies
- Include data, statistics, or research
- Embed relevant media (video, audio, slides)
- Link to authoritative sources
- Consider checklist or worksheet addition

**If Clarity scores <7**:
- Read aloud to check flow
- Replace jargon with simple terms
- Ensure brand voice consistency
- Check for contradictions with other content
- Have colleague review for clarity
