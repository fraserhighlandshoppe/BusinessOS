# Highland Business Operating System (HBOS)

HBOS is the specification and governance repository for evolving FrontAccounting into the business platform for Fraser Highland Shoppe.

## Repository Structure

- `00 Governance/` - Vision, Mission, Guiding Principles, Policies
- `01 Architecture/` - System architecture, business domains, technical overview
- `02 ADR/` - Architectural Decision Records (decisions, rationale, impacts)
- `03 Standards/` - Coding, API, Database, Security standards
- `04 Work Products/` - Inputs/Outputs, RACI charts, Deliverables, Templates
- `05 Backlog/` - Feature requests, User stories, Requirements

## Work Products

Project Management pages from the wiki (inputs/outputs, RACI charts, deliverables) should be integrated here as they become available. These documents describe:
- Work products and their inputs/outputs
- RACI matrices (Responsible, Accountable, Consulted, Informed)
- Deliverables and acceptance criteria
- Templates for consistent documentation

See Wiki Page Discovery Log for the complete list of pages to integrate.
