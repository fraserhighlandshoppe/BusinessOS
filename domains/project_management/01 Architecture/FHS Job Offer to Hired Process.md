# FHS Job Offer to Hired Process

## Overview
Process for onboarding new employees after job offer acceptance.

## Prerequisites
- Job offer presented and accepted
- Candidate has passed interview process
- See [[FHS Applicant Process]]

## Information Required

### Personal Information
- Name, Date of Birth
- Address and Phone Details
- Emergency Contact details
- Dependants

### Position Details
- Position (Job) details
- Salary Components
- Direct Deposit details (optional)
- Org Chart details

### Qualifications
- Educational qualifications
- Professional memberships
- Certifications
- Experience

## Onboarding Actions

### Step 1: Add to HRIS (OrangeHRM)
- Create employee record in OrangeHRM system
- Enter all personal and employment information

### Step 2: Add to IT Systems
1. **LDAP** - Add to directory services
2. **Workstations** - Configure workstation access
3. **OrangeHRM** - Create user for timesheets
4. **FrontAccounting** - Add with appropriate role permissions
5. **DotProject** - Add for project management access
6. **LimeSurvey** - Add for survey access
7. **phphlist** - Add for mailing list
8. **Wordpress** - Add for content management
9. **VTiger/SugarCRM/SuiteCRM** - Add for CRM access
10. **Moodle** - Add for learning management

### Day 1
- [ ] Meet team and management
- [ ] Review company policies

- [ ] IT system access verification
- [ ] Emergency procedures review

### Week 1
- [ ] Job shadowing
- [ ] System training
- [ ] Customer interaction training
- [ ] Product knowledge review

### Month 1
- [ ] Performance check-in
- [ ] Goal setting session

- [ ] Adjustment implementation
