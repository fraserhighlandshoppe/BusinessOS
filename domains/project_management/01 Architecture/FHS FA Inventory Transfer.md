# FHS FA Inventory Transfer

## Overview
Process for transferring inventory between warehouse locations in FrontAccounting.

## When to Use
- Moving stock between locations
- Warehouse reorganization
- Adjusting inventory locations
- Transfer from main store to event location

## Step-by-Step Process

### Step 1: Access Transfer Module
1. Log into FrontAccounting
2. Go to **Items and Inventory**
3. Click **Inventory Location Transfer**

### Step 2: Enter Transfer Details
**Inventory Location Transfer**:
- Creates transaction within item
1. Select **From** and **To** locations
2. Confirm **Date**
3. Confirm **Transfer Type** (typically Adjustment)
4. Enter Item Codes and Quantities
5. Click **ADD**
6. Enter Memo
7. Click **Process Transfer**

### Step 3: Print Transfer Document
On next page:
- Click **View this transfer** (to print)
- Print the transfer document

## After Transfer

### View Transaction
- Check transaction history
- Verify quantities transferred
- Confirm location update
