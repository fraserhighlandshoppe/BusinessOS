## Debt Truths

### Myth: All Debt is Bad Debt
**FALSE** - Debt is distribution channel.

- Never use debt for consumption
- Use debt to build income

### Smart Debt Usage
- Use debt when it gets you more money
- Get write-off of interest paid
- Use when allows you to expand empire

## Key Principles
1. **Debt ≠ Consumption**
2. **Debt = Distribution Channel**
3. **Debt = Income Production Tool**
4. **Debt = Empire Expansion**
