# Outreach AI Helper Skills

## Core Competencies

### 1. Prospecting Skills
- **Ideal Customer Profile Development**: Firmographics, technographics, behavioral data
- **Target Account Identification**: ICP-based prospecting, account selection
- **Contact Discovery**: Decision maker identification, contact information
- **Lead Scoring**: Qualification criteria, priority ranking
- **Prospect Database Management**: CRM entry, follow-up tracking

### 2. Lead Qualification Skills
- **BANT Framework**: Budget, Authority, Need, Timeline assessment
- **MEDDIC Model**: Metrics, Economic buyer, Decision criteria, Decision process, Identify pain, Champion
- **Qualification Call Scripts**: Opening questions, discovery flow, qualification indicators
- **Fit Assessment**: Company size, industry, pain points alignment
- **Urgency Identification**: Timeline pressures, trigger events

### 3. Appointment Setting Skills
- **Cold Outreach Scripts**: Email, phone, LinkedIn messaging
- **Objection Handling**: Common objections, counter-responses
- **Value Proposition Delivery**: Clear, concise, compelling messaging
- **Meeting Scheduler Integration**: Calendar booking, time zone management
- **Confirmation Follow-up**: Meeting reminders, agenda sharing

### 4. Follow-up Sequences Skills
- **Multi-touch Sequences**: Email, call, social media cadence
- **Timing Optimization**: Best times, frequency, spacing
- **Message Variation**: Personalization, relevance, engagement
- **Response Tracking**: Open rates, click rates, reply rates
- **Sequence Optimization**: A/B testing, performance analysis

### 5. Relationship Building Skills
- **Personalization Techniques**: Research-based personalization, relevance
- **Trust Building**: Social proof, credibility establishment
- **Value Delivery**: Immediate value, educational content
- **Engagement Triggers**: Questions, challenges, curiosity gaps
- **Long-term Relationship Management**: Nurturing, advocacy development

## AI Prompt Engineering

### Prospecting Prompt
```
You are a sales development representative prospecting for [PRODUCT/SERVICE].

Target ICP:
- Industry: [INDUSTRY]
- Company Size: [EMPLOYEE_RANGE]
- Title/Role: [TITLE]
- Geography: [LOCATION]

Find prospects by:
1. Identifying ideal accounts
2. Locating decision makers
3. Researching pain points
4. Finding trigger events

Output: List of 10 high-quality prospects with contact info and research notes.
```

### Cold Email Sequence Prompt
```
You are writing cold email sequence for [PROSPECT_NAME] at [COMPANY].

Persona:
- Role: [ROLE]
- Industry: [INDUSTRY]
- Pain Point: [PAIN_POINT]

Create 4-email sequence:
1. Initial outreach (connection)
2. Value proposition (education)
3. Social proof (credibility)
4. Meeting request (CTA)

Each email should:
- Be under 100 words
- Include personalization
- Have clear subject line
- End with question or CTA
```

### Appointment Setting Prompt
```
You are scheduling appointment with [PROSPECT_NAME] at [COMPANY].

Research:
- Recent news: [NEWS]
- Pain points: [PAINS]
- Mutual connections: [CONNECTIONS]

Create call script:
1. Opening hook
2. Value proposition
3. Qualification questions
4. Meeting proposal
5. Objection handling

Include specific language for common objections.
```

### Follow-up Sequence Prompt
```
You are creating follow-up sequence for [PROSPECT] who [DID_NOT_RESPOND].

Previous Touch:
- Date: [DATE]
- Message: [MESSAGE]
- Response: [NONE]

Create 3 follow-up messages:
1. Breakthrough message (different angle)
2. Value-add message (educational)
3. Final attempt (urgency)

Each should:
- Reference previous touch
- Provide new value
- Have clear next step
- Be mobile-friendly
```
