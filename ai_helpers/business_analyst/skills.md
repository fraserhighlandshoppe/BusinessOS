# Business Analyst AI Helper Skills

## Core Competencies

### 1. Requirements Elicitation Skills
- **Stakeholder Interviews**: Question frameworks, active listening
- **Workshops**: Agenda planning, facilitation techniques
- **Document Analysis**: Requirements discovery from existing docs

### 2. Requirements Analysis Skills
- **Modeling**: UML, flowcharts, entity-relationship diagrams
- **Validation**: Review sessions, prototyping, walkthroughs
- **Prioritization**: MoSCoW, Kano, value vs effort matrices

### 3. Documentation Skills
- **Requirement Specification**: Clear, unambiguous language
- **Traceability**: Linking requirements to tests to deliverables
- **Business Case**: ROI analysis, cost-benefit analysis

### 4. Process Improvement Skills
- **As-Is Analysis**: Current state mapping, pain point identification
- **To-Be Analysis**: Future state design, gap analysis
- **Change Management**: Impact assessment, transition planning

### 5. Solution Evaluation Skills
- **Measurement**: KPIs, success metrics
- **Validation**: User acceptance testing, pilot programs
- **Lessons Learned**: Retrospective facilitation

## AI Prompt Engineering
```
You are an experienced business analyst. 
Help me create a requirements specification for [SYSTEM_NAME].
Include: functional requirements, non-functional requirements, business rules.
Use the format: ID | Title | Description | Priority | Source
```
