# Quality Assurance AI Helper Skills

## Core Competencies

### 1. Test Planning Skills
- **Test Strategy**: Types of testing, entry/exit criteria
- **Test Plan**: Scope, schedule, resources, risks
- **Test Cases**: Pre-conditions, steps, expected results

### 2. Test Execution Skills
- **Manual Testing**: Exploratory testing, scenario testing
- **Automation**: Script creation, framework selection
- **Defect Management**: Bug reporting, triage, tracking

### 3. Quality Metrics Skills
- **Defect Density**: Calculation, interpretation
- **Test Coverage**: Statement, branch, path coverage
- **Process Metrics**: Cycle time, throughput, defect rate

### 4. Compliance Skills
- **Regulatory Requirements**: Industry standards, certifications
- **Audit Preparation**: Documentation, evidence gathering
- **Risk Assessment**: Security, privacy, compliance

### 5. Continuous Improvement Skills
- **Root Cause Analysis**: 5 Whys, Fishbone diagrams
- **Process Improvement**: Lean, Six Sigma concepts
- **Knowledge Sharing**: Lessons learned, best practices

## AI Prompt Engineering
```
You are a QA lead with 10+ years experience. 
Help me create test cases for [FEATURE_NAME].
Include: positive scenarios, negative scenarios, edge cases.
Format each test case as: ID | Title | Pre-conditions | Steps | Expected Result
```
