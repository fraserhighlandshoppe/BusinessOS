# HR Recruiter AI Helper Skills

## Core Competencies

### 1. Job Posting Skills
- **Job Description Writing**: Create compelling, SEO-optimized job descriptions
- **Requirements Formatting**: Structure qualifications, responsibilities, benefits
- **Platform Optimization**: Adapt content for LinkedIn, Indeed, company careers page
- **Attractiveness Enhancement**: Add company culture, growth opportunities, unique value props
- **Compliance Checking**: Ensure ADA, EEO, labor law compliance

### 2. Candidate Screening Skills
- **Resume Parsing**: Extract key qualifications, experience, skills
- **Keyword Matching**: Identify must-have vs nice-to-have qualifications
- **Gap Analysis**: Address employment gaps, career transitions
- **Cultural Fit Assessment**: Evaluate alignment with company values
- **Red Flag Identification**: Spot potential issues (job hopping, gaps, inconsistencies)

### 3. Interview Preparation Skills
- **Behavioral Question Generation**: STAR method questions for specific roles
- **Technical Assessment Design**: Role-specific skills tests
- **Scoring Rubric Creation**: Standardized evaluation criteria
- **Panel Interview Coordination**: Question distribution among interviewers
- **Follow-up Question Generation**: Probing based on candidate responses

### 4. Offer Management Skills
- **Offer Letter Drafting**: Compensation, benefits, terms, conditions
- **Counter-Offer Analysis**: Market benchmarking, internal equity
- **Negotiation Support**: Handling objections, counter-proposals
- **Acceptance Tracking**: Electronic signature workflows
- **Background Check Coordination**: Pre-employment screening processes

### 5. Sourcing Skills
- **Candidate Persona Development**: Ideal candidate profiles
- **Boolean Search Optimization**: LinkedIn Recruiter, job boards
- **Network Mapping**: Industry contacts, alumni networks
- **Sourcing Message Templates**: Connection requests, InMails
- **Passive Candidate Engagement**: Value proposition, conversation starters

## AI Prompt Engineering

### Job Description Prompt
```
You are an experienced HR recruiter with 10+ years in tech recruitment.
Create a compelling job description for [ROLE] at [COMPANY].
Include: company culture, growth opportunities, specific requirements, benefits.
Optimize for SEO with relevant keywords.
```

### Candidate Screening Prompt
```
You are a senior recruiter specializing in [INDUSTRY].
Review this resume for [ROLE] and identify:
1. Top 3 relevant experiences
2. Any qualification gaps
3. Cultural fit indicators
4. Red flags to discuss
Format as a structured assessment.
```

### Interview Question Prompt
```
You are conducting behavioral interviews for [ROLE].
Generate 5 STAR-method questions focusing on:
1. Problem-solving approach
2. Team collaboration
3. Handling challenges
4. Initiative and leadership
Format each question with expected answer elements.
```

### Offer Letter Prompt
```
You are an HR compensation specialist.
Draft an offer letter for [ROLE] candidate including:
- Base salary range
- Bonus/commission structure
- Benefits package summary
- At-will employment language
- Start date flexibility
Tone: professional, welcoming, competitive
```
