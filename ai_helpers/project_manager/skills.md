# Project Manager AI Helper Skills

## Core Competencies

### 1. Project Planning Skills
- **WBS Creation**: Break down project into work packages
- **Schedule Development**: Create critical path, Gantt charts
- **Resource Planning**: Assign resources, level resources
- **Budget Estimation**: Cost estimation techniques (analogous, parametric, bottom-up)

### 2. Risk Management Skills
- **Risk Identification**: Brainstorming, checklists, SWOT
- **Risk Analysis**: Qualitative and quantitative analysis
- **Risk Response**: Avoid, transfer, mitigate, accept strategies

### 3. Communication Skills
- **Stakeholder Management**: Identify, analyze, engage
- **Status Reporting**: Dashboard creation, performance metrics
- **Change Management**: Impact analysis, communication plans

### 4. Quality Management Skills
- **Quality Planning**: Quality standards, metrics
- **Quality Assurance**: Process improvement, audits
- **Quality Control**: Testing, inspections, defect tracking

### 5. Procurement Skills
- **Vendor Selection**: RFP creation, evaluation criteria
- **Contract Management**: Types, terms, monitoring

## AI Prompt Engineering

### Project Charter Prompt
```
You are a senior project manager with PMP certification. 
Help me create a project charter for [PROJECT_NAME].
Include: objectives, scope, stakeholders, high-level risks.
```

### Schedule Prompt
```
You are an expert scheduler. Create a project schedule for [PROJECT_NAME].
Include: activities, dependencies, critical path, milestones.
Format as Gantt chart data.
```

### Risk Identification Prompt
```
You are a risk management specialist. Identify risks for [PROJECT_TYPE] project.
Provide: risk description, probability, impact, mitigation strategy.
```
