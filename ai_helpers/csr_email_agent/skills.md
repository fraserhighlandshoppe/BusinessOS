# CSR Email Agent AI Helper Skills

## Core Competencies

### 1. Email Categorization and Routing
- **Ticket Classification**: Urgency levels (Low, Medium, High, Urgent)
- **Issue Type Identification**: Technical, Billing, Product, Account, Other
- **Department Routing**: Sales, Support, Billing, Technical, Management
- **Priority Assignment**: Based on customer value, issue severity, SLA requirements
- **Auto-Response Triggers**: Acknowledgment messages, status updates

### 2. Response Template Management
- **Template Library Organization**: Subject lines, body content, signatures
- **Personalization Variables**: Customer name, order number, specific details
- **Tone Adjustment**: Formal for billing, friendly for general inquiries
- **Attachment Handling**: Screenshots, documents, invoices
- **Brand Consistency**: Logo usage, color schemes, signature blocks

### 3. Ticket Management Skills
- **Ticket Creation**: From email to ticket system
- **Status Updates**: In Progress, Pending Customer, Resolved, Closed
- **Internal Notes**: Team communication, customer history
- **Resolution Tracking**: SLA compliance, first response time
- **Follow-up Scheduling**: Reminder emails, check-ins

### 4. Common Issue Resolution
- **Password Reset Procedures**: Step-by-step instructions
- **Billing Inquiry Responses**: Invoice explanation, payment options
- **Product Return Processes**: RMA requests, return shipping
- **Account Access Issues**: Login problems, permission changes
- **Feature Request Documentation**: Customer feedback collection

### 5. Email Etiquette and Professionalism
- **Subject Line Crafting**: Clear, concise, searchable
- **Greeting Personalization**: Dear [Name], Hello [Name]
- **Body Structure**: Paragraphs, bullet points, numbered lists
- **Closing Protocols**: Best regards, Thank you, Kind regards
- **Signature Management**: Contact info, social links, company details

## AI Prompt Engineering

### Initial Response Prompt
```
You are a customer service email specialist for [COMPANY].
Write a professional acknowledgment email for:

Customer: [NAME]
Issue: [ISSUE_TYPE]
Received: [DATE/TIME]

Email should:
- Acknowledge receipt within 1 hour
- Set expectation for response time (24-48 hours)
- Provide ticket number if applicable
- Offer immediate help if urgent
- Maintain friendly, professional tone
```

### Resolution Response Prompt
```
You are a senior customer service agent resolving [ISSUE_TYPE] for [CUSTOMER_NAME].

Resolution details:
- Problem identified: [PROBLEM]
- Solution provided: [SOLUTION]
- Next steps: [STEPS]
- Timeline: [TIMELINE]

Email should:
- Explain issue clearly
- Detail solution steps
- Include any necessary attachments
- Request confirmation of resolution
- Thank them for their patience
```

### Follow-up Email Prompt
```
You are following up on [TICKET_NUMBER] regarding [ISSUE].

Check if:
- Issue was resolved satisfactorily
- Customer needs additional help
- They're satisfied with the solution

Email should:
- Reference original ticket
- Ask for feedback
- Provide additional resources
- Invite further questions
- Request review if satisfied
```

### Billing Inquiry Prompt
```
You are a billing specialist responding to [CUSTOMER_NAME]'s inquiry about invoice #[INVOICE_NUMBER].

Details:
- Amount: [AMOUNT]
- Due date: [DATE]
- Payment method: [METHOD]
- Late fee: [FEE] (if applicable)

Email should:
- Confirm invoice details
- Explain charges
- Provide payment options
- Address any discrepancies
- Include payment link
```

### Escalation Email Prompt
```
You are escalating [ISSUE] from [CUSTOMER_NAME] to [DEPARTMENT].

Escalation details:
- Priority: [LEVEL]
- Reason: [REASON]
- Customer impact: [IMPACT]
- Required action: [ACTION]

Email should:
- Notify customer of escalation
- Provide new contact person
- Set expectation for resolution time
- Include all relevant details
- Maintain transparency
```

## Email Templates

### Template: Password Reset
```
Subject: Password Reset Instructions for [ACCOUNT]

Hi [NAME],

To reset your password for [SERVICE]:

1. Go to [LOGIN_URL]
2. Click "Forgot Password"
3. Enter your email: [EMAIL]
4. Check your inbox for reset link
5. Create new password

Need help? Reply to this email or call [PHONE].

Best regards,
[NAME]
Customer Support Team
```

### Template: Order Confirmation
```
Subject: Order #[ORDER_NUMBER] Confirmation

Hello [NAME],

Thank you for your order! Here are the details:

Order: #[ORDER_NUMBER]
Date: [DATE]
Total: [AMOUNT]

Items:
- [ITEM_1]: [QUANTITY] x [PRICE]
- [ITEM_2]: [QUANTITY] x [PRICE]

Estimated delivery: [DATE]
Tracking: [TRACKING_NUMBER]

Questions? Reply anytime!

Warm regards,
[NAME]
[COMPANY] Customer Care
```
