# Bookkeeper AI Helper Skills

## Core Competencies

### 1. Transaction Recording Skills
- **Daily Transaction Entry**: Sales, expenses, receipts, payments
- **Chart of Accounts Management**: Account creation, categorization
- **Journal Entry Preparation**: Adjusting entries, accruals, depreciation
- **Invoice Processing**: Customer and vendor invoices
- **Receipt Scanning and Categorization**: Digital receipt management

### 2. Bank Reconciliation Skills
- **Bank Statement Matching**: Transactions to general ledger
- **Outstanding Items Tracking**: Deposits in transit, outstanding checks
- **Bank Fee Recording**: Service charges, wire fees, NSF items
- **Reconciliation Discrepancy Investigation**: Missing transactions, errors
- **Monthly Reconciliation Reports**: Clean reconciliation statements

### 3. Financial Reporting Skills
- **Income Statement Preparation**: Revenue, expenses, net income
- **Balance Sheet Compilation**: Assets, liabilities, equity
- **Cash Flow Statement**: Operating, investing, financing activities
- **Account Analysis**: Aging reports, variance analysis
- **Budget vs. Actual Reporting**: Performance tracking

### 4. Accounts Management Skills
- **Accounts Receivable Tracking**: Customer balances, aging
- **Accounts Payable Management**: Vendor payments, due dates
- **Inventory Valuation**: FIFO, LIFO, weighted average
- **Fixed Asset Tracking**: Depreciation schedules, disposals
- **Loan and Credit Management**: Principal, interest tracking

### 5. Compliance and Tax Skills
- **Sales Tax Calculation**: Rates, collection, remittance
- **Payroll Tax Filing**: W-2s, 1099s, quarterly filings
- **Tax Preparation Support**: Schedule C, Form 1040
- **Audit Preparation**: Documentation, compliance checks
- **Regulatory Reporting**: Industry-specific requirements

## AI Prompt Engineering

### Transaction Entry Prompt
```
You are a bookkeeper entering transactions for [DATE].

Transactions to record:
1. [DESCRIPTION] - [AMOUNT] - [ACCOUNT]
2. [DESCRIPTION] - [AMOUNT] - [ACCOUNT]
3. [DESCRIPTION] - [AMOUNT] - [ACCOUNT]

For each transaction, provide:
- Date
- Account affected
- Debit/Credit amount
- Reference number
- Memo/description

Format as journal entries ready for posting.
```

### Bank Reconciliation Prompt
```
You are reconciling bank statement for [PERIOD].

Bank Statement Balance: [BANK_BALANCE]
Book Balance: [BOOK_BALANCE]

Outstanding Items:
- Deposits in Transit: [DEPOSITS]
- Outstanding Checks: [CHECKS]
- Bank Errors: [ERRORS]

Prepare reconciliation worksheet showing:
1. Starting balance
2. Additions to book
3. Deductions from book
4. Adjusted book balance
5. Explanation of differences
```

### Financial Statement Prompt
```
You are preparing financial statements for [PERIOD].

Trial Balance:
[TRIAL_BALANCE_DATA]

Generate:
1. Income Statement (Profit & Loss)
2. Balance Sheet
3. Cash Flow Statement (if applicable)

Include:
- All relevant line items
- Proper formatting
- Comparative periods (if available)
- Key ratios (if requested)
```

### Adjusting Entry Prompt
```
You are preparing adjusting entries for [PERIOD].

Adjustments needed:
1. Accrued Expenses: [DESCRIPTION] - [AMOUNT]
2. Prepaid Expenses: [DESCRIPTION] - [AMOUNT]
3. Depreciation: [ASSET] - [AMOUNT]
4. Accrued Revenue: [DESCRIPTION] - [AMOUNT]

Prepare adjusting journal entries with:
- Date
- Account titles
- Debit/Credit amounts
- Explanation
```

### Month-End Close Prompt
```
You are performing month-end close procedures for [MONTH/YEAR].

Tasks:
1. Review all transactions for period
2. Complete bank reconciliations
3. Record adjusting entries
4. Post all entries
5. Run financial statements
6. Review account balances
7. Prepare closing entries
8. Update opening balances

Provide month-end close checklist with completion dates.
```

## Bookkeeping Templates

### Template: Journal Entry
```
Date: [DATE]
Account: [ACCOUNT]
Debit: [DEBIT]
Credit: [CREDIT]
Reference: [NUMBER]
Memo: [DESCRIPTION]

Explanation: [BRIEF_EXPLANATION]
```

### Template: Bank Reconciliation
```
Bank Statement Balance: [AMOUNT]
Add: Deposits in Transit: [AMOUNT]
Less: Outstanding Checks: [AMOUNT]
Bank Service Charges: [AMOUNT]
Adjusted Bank Balance: [AMOUNT]

Book Balance: [AMOUNT]
Add: Credits: [AMOUNT]
Less: Debits: [AMOUNT]
Adjusted Book Balance: [AMOUNT]

Difference: $0.00 ✓
```

### Template: Invoice Entry
```
Date: [DATE]
Customer: [NAME]
Invoice #: [NUMBER]
Terms: [TERMS]
Due Date: [DATE]

Line Items:
1. [ITEM] x [QTY] @ [RATE] = [AMOUNT]
2. [ITEM] x [QTY] @ [RATE] = [AMOUNT]

Subtotal: [AMOUNT]
Tax: [AMOUNT]
Total: [AMOUNT]

Accounts: Accounts Receivable, Sales Revenue
```

### Template: Expense Entry
```
Date: [DATE]
Vendor: [NAME]
Category: [EXPENSE_TYPE]
Payment Method: [METHOD]

Description: [DESCRIPTION]
Amount: [AMOUNT]

Accounts: [EXPENSE_ACCOUNT], [CASH/BANK_ACCOUNT]
Receipt: [ATTACHED]
```
