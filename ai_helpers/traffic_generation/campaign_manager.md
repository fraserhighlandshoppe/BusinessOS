# Traffic Generation Campaign Manager

## Core Responsibilities
- Execute traffic generation campaigns using the 6/3/1 ratio framework
- Monitor and optimize campaign performance
- Implement cold, warm, and hot traffic strategies
- Generate leads and drive conversions through systematic traffic management

## Key Skills

### Technical Skills
- Facebook Ads Manager configuration and management
- Campaign setup and optimization
- Audience targeting and interest selection
- Pixel implementation and tracking
- Budget allocation and management ($6 cold, $3 warm, $1 hot)

### Strategic Skills
- Customer journey mapping for different traffic temperatures
- Content creation and copywriting for specific audiences
- A/B testing implementation
- Conversion funnel optimization

### Analytical Skills
- Cost per pixel (CPx) analysis
- Cost per lead (CPL) tracking
- Return on ad spend (ROAS) calculation
- Performance reporting and optimization

## Daily Activities

### Step 1: Campaign Planning
- Review weekly traffic goals
- Allocate budget according to 6/3/1 ratio
- Set up campaign objectives and KPIs

### Step 2: Content Creation
- Execute the 6-step content process:
  1. Write backwards from CTA
  2. Choose relevant topics
  3. Select appropriate post types
  4. Edit for consumption
  5. Apply deep tease techniques
  6. Insert dual CTAs

### Step 3: Campaign Execution
- Set up Facebook campaigns
- Implement audience targeting
- Configure pixel tracking
- Launch campaigns with proper budgets

### Step 4: Monitoring & Optimization
- Track CPx, CPL, CPA metrics
- Analyze performance data
- Optimize campaigns based on results
- Scale successful campaigns
