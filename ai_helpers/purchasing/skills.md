# Purchasing AI Helper Skills

## Core Competencies

### 1. Supplier Evaluation Skills
- **Supplier Qualification**: Financial stability, capacity, quality systems
- **RFP/RFQ Creation**: Requirements definition, evaluation criteria
- **Vendor Scoring**: Weighting factors, comparative analysis
- **Reference Checking**: Customer references, performance verification
- **Risk Assessment**: Financial, operational, geopolitical risks

### 2. Procurement Strategy Skills
- **Category Strategy**: Commodity analysis, sourcing strategy
- **Spend Analysis**: Spend visibility, consolidation opportunities
- **Contract Strategy**: Terms negotiation, compliance requirements
- **Sourcing Events**: RFP/RFQ management, bid evaluation
- **Strategic Sourcing**: Long-term partnerships, category management

### 3. Contract Management Skills
- **Contract Review**: Terms analysis, risk identification
- **Renewal Management**: Expiration tracking, negotiation timing
- **Compliance Monitoring**: SLA tracking, performance metrics
- **Amendment Processing**: Change orders, addendums
- **Dispute Resolution**: Issue identification, escalation paths

### 4. Cost Analysis Skills
- **Total Cost of Ownership**: Acquisition, operation, disposal costs
- **Price Benchmarking**: Market pricing, negotiation support
- **Cost Reduction Identification**: Waste elimination, efficiency gains
- **Supplier Pricing Negotiation**: Volume discounts, terms optimization
- **ROI Analysis**: Investment justification, payback calculation

### 5. Purchase Order Management Skills
- **PO Creation**: Requisition processing, approval workflows
- **PO Modification**: Change orders, cancellation procedures
- **PO Tracking**: Status monitoring, delivery scheduling
- **PO Reconciliation**: Invoice matching, three-way matching
- **Supplier Communication**: Order confirmations, status updates

## AI Prompt Engineering

### Supplier Evaluation Prompt
```
You are evaluating supplier for [PRODUCT/SERVICE].

Supplier Information:
- Company: [COMPANY]
- Years in Business: [YEARS]
- Location: [LOCATION]
- Capacity: [CAPACITY]
- Certifications: [CERTIFICATIONS]

Evaluate against criteria:
1. Financial Stability (30%)
2. Quality Systems (20%)
3. Delivery Performance (20%)
4. Pricing Competitiveness (15%)
5. Innovation Capability (10%)
6. Sustainability (5%)

Provide weighted score and recommendation with risk factors.
```

### Procurement Strategy Prompt
```
You are developing procurement strategy for [CATEGORY].

Category Details:
- Spend: [AMOUNT]
- Suppliers: [SUPPLIER_COUNT]
- Criticality: [HIGH/MEDIUM/LOW]
- Market: [MARKET_DESCRIPTION]

Develop strategy including:
1. Sourcing approach (single/dual/multi)
2. Supplier base optimization
3. Risk mitigation plan
4. Cost reduction initiatives
5. Technology enablement

Output: Procurement strategy document with action plan.
```

### Purchase Order Creation Prompt
```
You are creating purchase order for [ITEM] from [SUPPLIER].

PO Details:
- Item: [ITEM_DESCRIPTION]
- Quantity: [QUANTITY]
- Unit Price: [PRICE]
- Total: [AMOUNT]
- Required Date: [DATE]
- Delivery Location: [LOCATION]

Generate PO with:
1. Header information
2. Line item details
3. Terms and conditions
4. Approval routing
5. Tracking numbers

Output: PO ready for approval with reference numbers.
```

### Cost Analysis Prompt
```
You are analyzing total cost of ownership for [PRODUCT/SERVICE].

Cost Components:
- Purchase Price: [AMOUNT]
- Shipping: [AMOUNT]
- Installation: [AMOUNT]
- Operating: [ANNUAL_AMOUNT]
- Maintenance: [ANNUAL_AMOUNT]
- Disposal: [AMOUNT]

Calculate:
1. Total cost over [PERIOD]
2. Annualized cost
3. Cost per unit
4. Comparison to alternatives
5. ROI if applicable

Provide cost analysis with recommendations.
```
